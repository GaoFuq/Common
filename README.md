## 功能介绍
1、base包中提供了 BaseLazyFragment、BaseDialogFragment。

2、提供了很多常用工具类、自定义View。

3、常用第三方库混淆的添加。

## 使用方法：

1、引用

在Project的gradle中加入：
```groovy
    allprojects {
        repositories {
            ...
            maven { url 'https://jitpack.io' }
        }
    }
```
在Module的gradle中加入：
```groovy
    dependencies {
        compile 'com.github.like5188:Common:版本号'
    }
```
